Colors

The MNR Framework color palette consists of three colors defined in the _colors.scss file.

By default, those colors are turquoise, white, and black. There are also default colors set for several other elements, including buttons and alerts. All of these colors can be changed by altering these variables. 


    $mainColor: #fff;
    $secondaryColor: #000;
    $accentColor: #4bc4cd;
    $warningColor: #F4CDA5;
    $successColor: #5DB5A4;
    $infoColor: #5bacff;
    $dangerColor: #ED5276;
    $primaryColor: #183658;
