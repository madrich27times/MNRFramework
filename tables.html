Notice that tables are wrapped in divs, and the "table" tag is applied to that div.

You can get the basic table by using the "table-basic" class tag. 


    <div class="table-basic">
      <table>
        <tr>
          <th>Heading 1</th>
          <th>Heading 2</th>
          <th>Heading 3</th>
          <th>Heading 4</th>
        </tr>
