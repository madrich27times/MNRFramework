Buttons

Several buttons are included in the MNR Framework:
  • Option 1
  • Option 2
  • Option 3

Each button comes equipped with hover effects and is responsive by default.

All of the buttons are also fully customizable through the UI-Elements file in the framework. You can also alter their default colors through the _colors.scss file. 

 
    <button class="btn-basic">Basic</button>
    <button class="btn-primary">Primary</button>
    <button class="btn-info">Info</button>
    <button class="btn-success">Success</button>
    <button class="btn-warning">Warning</button>
    <button class="btn-danger">Danger</button>
    <button class="btn-arrow">
      <span>Arrow </span>
    </button>
    <div class="dropdown">
    <button class="dropdown-btn">Dropdown</button>
      <ul class="dropdown-content">
        <li>Dropdown content</li>
        <li>Dropdown content</li>
        <li>Dropdown content</li>
      </ul>
    </div>

You can change the default buttons' colors by changing the color variables in the _colors.scss file. 


    $warningColor: #F4CDA5;
    $successColor: #5DB5A4;
    $infoColor: #5bacff;
    $dangerColor: #ED5276;
    $primaryColor: #183658;
