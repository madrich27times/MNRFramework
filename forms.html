Forms

Forms in the MNR Framework were developed mobile-first, meaning they will scale to any size without breaking.

Go ahead, resize the browser!

Notice that this form is wrapped in a div. The form will resize to fill its parent element because it has a width of 100% by default.

Just include the "form" class tag! 


    <div class="padded-div">
      <form class="form">
      <label for="fname">First Name</label>
      <input type="text" id="fname" name="firstname" placeholder="First name">

      <label for="lname">Last Name</label>
      <input type="text" id="lname" name="lastname" placeholder="Last name">

      <label for="email">Email</label>
      <input type="text" id="email" name="email" placeholder="Email">

      <label for="phone">Phone Number</label>
      <input type="text" id="phone" name="phone" placeholder="Phone">

      <label for="select">Select</label>
      <select id="select" name="select">
        <option value="op1">Option 1</option>
        <option value="op2">Option 2</option>
        <option value="op3">Option 3</option>
      </select>

      <input type="submit" value="Submit">
      </form>
    </div>

You can alter the color of the form border by changing the accentColor variable in _colors.scss.


    $accentColor: #4bc4cd;
