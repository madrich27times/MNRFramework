Custom Javascript Dropdown List

The MNR Framework provides a custom class for creating your own styled dropdown lists with ease.

Dropdown List

    <div id="dd" class="dropdown-list" tabindex="1">
      <span>Dropdown List</span>
        <ul class="dropdown-list-contents">
          <li>
            <a href="#">Link</a>
          </li>
          <li>
            <a href="#">Link</a>
          </li>
          <li>
            <a href="#">Link</a>
          </li>
        </ul>
    </div>

You can change the color of the arrow in the list by changing the accentColor variable in _colors.scss. 


    $accentColor: #4bc4cd;

Be sure to include the JavaScript library at the bottom of your html page! 


    <script src="scripts/mnr_script.js"></script>
