Parallax

Parallax is one of the biggest trends in website design today so the MNR Framework has made it super easy to integrate into your website!

View Parallax

Add parallax in two simple steps!

Create your layout by including the "parallax" class tag and "parallax_" id on your div elements.

You must make a separate element for each image you wish to display with the corresponding id number. Ids must also start at 0. 


    <div class="parallax" id="parallax_0"></div>
    <div class="parallax" id="parallax_1"></div>
    <div class="parallax" id="parallax_2"></div>

Edit the "parallax-pictures.json" file to hold the images you wish to use in your site.

You must include the full path to the image you wish to use as well as the file extension! 


    {
      "pictures": [
        "images/your_image.png",
        "images/your_other_image.png",
        "images/your_other_other_image.png"
      ]
    }

Be sure to include the JavaScript library at the bottom of your html page! 


    <script src="scripts/mnr_script.js"></script>
