Custom Classes

The MNR Framework provides you with several useful custom classes to make developing and styling fast and simple!

Cards

Quickly organize information into clean, easy to read cards using the "card" tag. Cards consist of a header and body section wrapped in a containing div, and are responsive when scaling.

Image Cards

Image cards allow you to add a clean, simple description to any image by adding the "img-card" class to a div.

Description


    <div class="center-block">
      <div class="img-card">
        <img src="images/img_url.png">
        <div class="img-card-container">
          <p>Your Description</p>
        </div>
      </div>
    </div>

Tooltips

Tooltips are a quick and fun way to provide your users with more information. The MNR Framework provides top, bottom, left, and right tooltips with customizable colors.

  • Tooltip Right Tooltip text
  • Tooltip Left Tooltip text
  • Tooltip Up Tooltip text
  • Tooltip Down Tooltip text 

    <div class="tooltip">Tooltip Right
      <span class="tooltip-right">Tooltip text</span>
    </div>

    <div class="tooltip">Tooltip Left
      <span class="tooltip-left">Tooltip text</span>
    </div>

    <div class="tooltip">Tooltip Up
      <span class="tooltip-top">Tooltip text</span>
    </div>

    <div class="tooltip">Tooltip Down
      <span class="tooltip-bottom">Tooltip text</span>
    </div>

You can change the color of the tooltip background by changing the accentColor variable in _colors.scss. You can also change the color of the tooltip text by changing the mainColor variable in the same file. 


    $mainColor: #fff;  
    $accentColor: #4bc4cd;

Alerts

Use the alert class to display a message to your users. Similar to buttons, alerts in the MNR Framework come in several default colors, all of which can be altered to fit your needs.

× Basic alert
× Info alert
× Warning alert
× Success alert
× Danger alert 

    <div class="alert-basic">
      <span class="close-btn" onclick="this.parentElement.style.display='none';">×</span>
      Basic alert
    </div>
    <div class="alert-info">
      <span class="close-btn" onclick="this.parentElement.style.display='none';">×</span>
      Info alert
    </div>
    <div class="alert-warning">
      <span class="close-btn" onclick="this.parentElement.style.display='none';">×</span>
      Warning alert
    </div>
    <div class="alert-success">
      <span class="close-btn" onclick="this.parentElement.style.display='none';">×</span>
      Success alert
    </div>
    <div class="alert-danger">
      <span class="close-btn" onclick="this.parentElement.style.display='none';">×</span>
      Danger alert
    </div>

You can change the default alert colors by changing the color variables in the _colors.scss file. 


    $accentColor: #4bc4cd;
    $warningColor: #F4CDA5;
    $successColor: #5DB5A4;
    $infoColor: #5bacff;
    $dangerColor: #ED5276;
    $primaryColor: #183658;
