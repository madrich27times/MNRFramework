Modals

Notice that the id of the modal container matches the data-modal attribute of the button to open the modal.

These fields must be unique on a single page and match one another if you want to be able to see your beautiful modal pop up! 


    <button class="btn-modal" data-modal="modalLight">Open Modal</button>
      <div id="modalLight" class="modal modal-light">
        <div class="modal-content">
          <div class="modal-header">
            <span class="close">×</span>
            <h4>Header</h4>
          </div>
          <div class="modal-body">
            <p class="tagline">Content</p>
          </div>
          <div class="modal-footer">
            <h5>Footer</h5>
         </div>
      </div>
    </div>

You can change the color of the modal border by changing the accentColor variable in the _colors.scss file. 


    $accentColor: #4bc4cd;

You can also use the default dark theme modal with the "modal-dark" class tag.
