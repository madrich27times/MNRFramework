Message of the Day

Add a "message of the day" feature to your website easily with the MNR Framework.

× 

    <div class="message-otd">
      <span class="close-btn" onclick="this.parentElement.style.display='none';">×</span>
    </div>

You can change the color of the message background by changing the accentColor variable in _colors.scss.

The message uses the complement of the accentColor variable. 


    $accentColor: #4bc4cd;

You can also change the color of the message text by changing the mainColor variable in _colors.scss. 


    $mainColor: #fff;

You can change the content of the message easily by altering the message.json file in the JavaScript library.

If you don't want your message to include a date, be sure to set the "date" field to false. 


    {
      "date": "true",
      "message": "Welcome! Today is "
    }

Be sure to include the JavaScript library at the bottom of your html page! 


    <script src="scripts/mnr_script.js"></script>
