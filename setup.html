Getting Started

Getting started with the MNR Framework is easy. Just choose your layout, and start building!

You can also customize the default color palette and fonts to make your site more unique to your own style.

Change Colors! Change Fonts!

Choosing Layout

This is probably the hardest part of building any website. Luckily, the MNR Framework provides several adaptable layouts to fit your design needs. And the best part is, they're all responsive by default!

The recommended layout is the Three Column layout, which is used by this site.

The three column layout uses a sticky header and footer to condense your site depending on the amount of content. It scales down to a single column layout with a horizontal nav bar.

Resize the browser to see the responsiveness in action! 


    <body class="three-col-container">
      <header class="site-header">Logo and site name</header>
      <div class="three-col-body">
        <main class="three-col-content">The main content of your site</main>
        <nav class="three-col-nav">Navigation controls</nav>
        <aside class="three-col-aside">Ads or other information</aside>
      </div>
      <footer class="site-footer">Footer information</footer>
    </body>

The third column in the layout, the aside, is optional, as this website displays. The rest of the layout will remain responsive without it.

This site uses an extra tag on the header element to align the logo and the website name nicely. If your header includes an image and text, use the "logo-header" class and the "img-logo-inline" class on your image. 


    <header class="site-header logo-header">
      <img src="images/img_url.png" class="img-logo-inline">
      <h4>Your Site Name</h4>
    </header>

You may also choose to use the row layout, which is a more traditional website layout consisting of a header with nav, the main content, and the footer.

The row layout also scales down to a single column layout.

Other Layout Elements

Regardless of which layout you choose, there are still more ways to customize your layout elements!

Navs

While it is recommended that you use the navbar class that corresponds to the layout you are using, the custom classes to style your navbar will work in either layout.

This site uses the "nav-light-color" class, which is based off of the "nav-light" class. It includes colored links on hover.

  • Link2
  • Link3
  • Link4 

    <nav class="nav-light-color"></nav>

The "nav-light-color" link colors can be modified by changing the color variables in the _colors.scss file.

The secondaryColor variable controls the natural and visited state of the link, while the complement of the accentColor variable controls the active and hover state of the link. 


    $mainColor: #fff;
    $secondaryColor: #000;
    $accentColor: #4bc4cd;

If you prefer a more neutral look for your navbar elements, you can use the "nav-light" class, which includes a white background with a grey hover effect on navbar elements.
