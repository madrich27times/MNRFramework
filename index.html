Welcome to the MNR Framework

Inspired by Bootstrap, the MNR Framework is powered by SASS and Flexbox. Our goal is to create a versatile, maintainable CSS framework to meet all of your project needs.

The framework is constantly being updated, so check back often to keep up with all of the new features that are readily available!
