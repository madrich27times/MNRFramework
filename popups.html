Popups

Similar to the tooltips in the MNR Framework, popups are a fun custom class to increase your user experience!

  • Click here! Popup content 

    <div class="popup">Click here!
      <span class="popup-content">Popup content</span>
    </div>

You can change the color of the popup background by changing the accentColor variable in _colors.scss. 


    $accentColor: #4bc4cd;

You can also change the color of the popup text by changing the mainColor variable in _colors.scss. 


    $mainColor: #fff;

Be sure to include the JavaScript library at the bottom of your html page! 


    <script src="scripts/mnr_script.js"></script>
