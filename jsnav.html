Dropdown Navbar

Add a dropdown to your navbar easily in the MNR Framework.

Sidenav dropdown bar using the three-column layout (used on this page!). 


    <nav class="nav-light-color three-col-nav-dropdown">
      <ul class="nav-list">
        <li>
          <a href="index.html">Home</a>
        </li>
      </ul>
      <button class="dropdown-nav-btn">Button
        <i class="fa-caret"></i>
      </button>
      <div class="dropdown-nav-container">
        <ul>
          <li>
            <a href="link.html">Hidden Link</a>
          </li>
        </ul>
      </div>
    </nav>

    <nav class="navbar-fixed-col-dropdown nav-light-color">
      <a href="#">Link</a>

      <a href="#">Link4</a>
      <div class="dropdown-nav-wrapper">

          <i class="fa-caret"></i>
        </button>
        <div class="dropdown-nav-container">
          <ul>
            <li>
              <a href="#">Link5</a>
            </li>
            <li>
              <a href="#">Link6</a>
            </li>
          </ul>
        </div>
      </div>
    </nav>

Be sure to include the JavaScript library at the bottom of your html page! 


    <script src="scripts/mnr_script.js"></script>
