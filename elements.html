Elements

Several elements in the MNR Framework have been customized to make development fast and easy by setting default style values.

Headings use the Raleway font by default, which can be changed by changing the headingFont variable in _fonts.scss


    $headingFont: 'Raleway', sans-serif;

Links

By default, links have no text-decoration. Their hover and active states are set to the accentColor, while their natural and visited states are set to the secondaryColor.

I am a link!

The default colors for links can be changed in the _colors.scss file. 


    $secondaryColor: #000;
    $accentColor: #4bc4cd;

Paragraphs

Paragraphs use Flexbox to become responsive at different browser sizes. They have a margin of 10px by default.

Tagline

The tagline class is a custom element class that gives emphasis to certain inline text elements. It has been applied to paragraphs throughout this site, but it can also be used with spans to give text a slightly larger font size and an increaed margin.

Pay attention to me, I am a tagline element! 

    <span class="tagline">Pay attention to me, I am a tagline element!</span>

Divs

All divs use Flexbox to maintain responsiveness with a default flex value of 1.

Padded-Div

The padded-div-struct class is a custom element class that gives div elements extra padding and margins. It is intended to be used to create sections in your layout.

Center

The center class is a custom element class that centers elements inside of other elements.

For example, here is a button centered within a parent div element. 


    <div class="center">
    <button class="btn-primary">Button</button>
    </div>

Blockquotes

The default styling on blockquote elements is rather bland, so the MNR Framework has fixed that by creating clean, colorful blockquotes by default. Simply use the blockquote tag as you normally would and enjoy the inherited styling!

Here is a blockquote element! Lorem ipsum dolor sit amet, sit quam donec leo vivamus urna nam, quam dignissim, donec enim ipsum, amet erat, convallis condimentum potenti fusce. Sit etiam non pellentesque eros condimentum. Velit dui nec ut sit diam. Sed habitasse commodo vitae, nunc ipsum reprehenderit.

You can change the background color of blockquote elements by changing the accentColor variable in the _colors.scss file. 


    $accentColor: #4bc4cd;

Horizontal Rules

Horizontal rules in the MNR Framework have been stylized to match the color theme of your website.

You can change the color of horizontal rules by changing the accentColor variable in the _colors.scss file. 


    $accentColor: #4bc4cd;

Inline-List

The inline-list class is a custom element class that can be used to align lists horizontally without having to struggle with setting floats. Simply add the class tag to your list and you're done!
